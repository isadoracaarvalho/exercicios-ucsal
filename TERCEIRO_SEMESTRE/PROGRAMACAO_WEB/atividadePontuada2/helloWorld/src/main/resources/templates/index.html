Informações gerais sobre o projeto

Essa aplicação simples foi desenvolvida com Spring Boot e tem como objetivo principal exibir informações sobre usuários, que incluem nome, email e telefone.

Funcionalidades

  • Página inicial: É responsável por apresentar uma visão geral do projeto e suas principais funcionalidades.
  • Home: Detalha o funcionamento do sistema e a navegação pelas páginas.
  • API de usuários: Endpoint do tipo GET, que apresenta uma lista dos usuários em formato JSON.
