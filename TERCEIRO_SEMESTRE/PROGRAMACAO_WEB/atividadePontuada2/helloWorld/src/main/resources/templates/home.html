Esse espaço é destinado a explicações sobre o funcionamento do sistema e as opções que ele ofereçe.

Funcionalidades Detalhadas

  • Exibição de Usuários: O sistema mantém uma lista de 4 usuários fictícios, com os detalhes de cada um (nome, e-mail, telefone).
  • API GET: O endpoint /projeto/getAll retorna todos os usuários cadastrados em formato JSON. Você pode acessar diretamente em /projeto/getAll

Tecnologias Usadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

  • Spring Boot: Para criação da API e lógica de backend.
  • Thymeleaf: Para renderização de páginas HTML dinâmicas.
  • CSS: Para estilização da interface do usuário.
  • H2 Database: Banco de dados relacional leve e em memória, utilizado para armazenar temporariamente os dados dos usuários durante a execução da aplicação. O banco pode ser acessado através do console web em /h2-console, facilitando a visualização e manipulação dos dados diretamente pelo navegador.
